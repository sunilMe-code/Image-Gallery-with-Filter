🖼️ Image Gallery with Filter

A responsive and interactive Image Gallery with Filter built using HTML, CSS, and Vanilla JavaScript. This project allows users to filter images dynamically based on categories with smooth UI interactions.

The application demonstrates how to use JavaScript DOM manipulation, event handling, and filtering logic to create modern frontend features commonly used in portfolios and gallery websites.

This project is ideal for beginners looking to improve their frontend development skills by building practical and visually appealing UI components.

✨ Features

🖼️ Dynamic image gallery

🔍 Filter images by category

⚡ Smooth transitions & animations

🎨 Clean and modern UI

📱 Fully responsive design

🛠️ Tech Stack

HTML5 – Structure

CSS3 – Styling

JavaScript (ES6) – Filtering logic
